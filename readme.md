### Installation
#### Set your environment variables
`TOKEN`  - The Discord Bot Token
#### Install the requirements
`pip install -r requirements.txt`


### Launch the bot
`python -m bot`

### Commands
`!leaderboard` - Fetches the top 10 leaderboard from The Funded Trader
